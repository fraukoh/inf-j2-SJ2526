Aufgabe: Gewinnspiel auf dem Jahrmarkt

Auf einem Jahrmarkt gibt es ein Gewinnspiel mit Losen. Jedes Los hat eine Nummer. Die Gewinne werden nach folgenden Regeln verteilt:

  • Wer die Losnummer 1000, 2000 oder 3000 hat, gewinnt ein Theater-Ticket
  • Wer eine Losnummer über 2000 hat (außer 3000), gewinnt ein Plüschtier
  • Wer eine Losnummer zwischen 1001 und 2000 hat, gewinnt einen Radiergummi
  • Alle anderen Losnummern gewinnen nichts

Schreiben Sie ein Programm, das eine Losnummer einliest und den entsprechenden Gewinn ausgibt.

Lösung

▼ Lösung anzeigen
losnr = int(input("Bitte die Losnummer eingeben:"))
if losnr == 1000 or losnr == 2000 or losnr == 3000:
    print("Theater-Ticket")
else:
    if losnr > 2000:
        print("Plüschtier")
    else:
        if losnr > 1000:
            print("Radiergummi")
        else:
            print("Kein Gewinn")
